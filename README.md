# Bunny Dash: Forest Escape

A retro arcade-style chase game built for IST 130 (Spring 2026) at Penn State.

Help **Clover** the bunny race along winding dirt paths and collect carrots before the fox catches up. Stay on the path — the grass slows you down.

**Play:** open `index.html` in any modern browser, or use the GitHub Pages link.

**Controls**

- Arrow keys — move
- M — mute music
- R — restart current level
- Space — advance through the intro / between levels

Five levels, escalating speed and path complexity. Self-contained — one HTML file, no external assets.
